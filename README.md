# 🐕 寵物年齡計算機

## 📝 專案介紹
寵物年齡計算機是一個簡單實用的網頁應用，幫助您快速計算寵物的實際年齡。

## ✨ 主要功能
- 輸入寵物的出生年月日
- 自動計算寵物的實際年齡（以歲為單位）
- 將寵物年齡換算成人類年齡
- 直觀易用的使用者介面

## 🎯 使用方式
1. 在網頁上輸入寵物的出生年月日
2. 點擊計算按鈕
3. 立即查看寵物的年齡和對應的人類年齡

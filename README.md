# 狗狗年齡換算器

## 產品規格

1. 使用者可以輸入狗狗的出生日期。
2. `日期欄位`的最大值為今日，避免選擇未來日期。
3. 使用者按下「🐶 計算年齡」按鈕後，會計算並顯示：
   - 狗狗實際年齡（歲 + 月）
   - 對應的人類歲數
   - 生命階段說明
4. 若未選擇日期或選擇未來日期，會顯示錯誤訊息。
5. 支援按 Enter 鍵觸發計算功能。

## localStorage 功能

1. 使用者選擇日期後，會把選過的日期儲存到 `localStorage`。
2. 下次載入頁面時，會自動從 `localStorage` 讀取先前輸入的日期，並帶回到日期欄位中。
3. 若先前日期有效，頁面載入時會自動執行一次計算，並顯示對應結果。

## 計算邏輯

- 狗狗實際年齡：使用 `現在時間 - 出生日期`，並以 `365.25 天` 換算成年。
- 狗狗年齡顯示為整數歲與對應的月數。
- 人類年齡：
  - 若狗狗年齡小於 0.08 年（約 1 個月），採用週數估算。
  - 否則使用公式：`16 × ln(狗齡) + 31`。
